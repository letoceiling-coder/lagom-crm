# Обновление документации и меню админ-панели

## Выполненные изменения

### 1. Добавлен раздел SEO в документацию

**Файл:** `resources/js/pages/admin/Documentation.vue`

Добавлен новый раздел "SEO оптимизация" с подробным описанием:

#### Содержание раздела:
- **Глобальные SEO настройки** - как настроить базовые параметры SEO
- **SEO для отдельных страниц** - настройка SEO для продуктов, услуг, кейсов и пользовательских страниц
- **Автоматические SEO инструменты** - описание robots.txt, sitemap.xml, canonical URL, Schema.org
- **Рекомендации по SEO** - лучшие практики и советы
- **Проверка индексации** - инструкции по работе с Google Search Console и Яндекс.Вебмастер

#### Раздел добавлен в навигацию документации:
- Пункт "SEO оптимизация" добавлен в список разделов
- Доступен через боковое меню документации

### 2. Добавлен статус модулей в меню

**Файл:** `app/Services/AdminMenu.php`

Добавлено поле `module_connected` для каждого пункта меню:
- `true` - модуль подключен и работает
- `false` - модуль не подключен

#### Модули с `module_connected => false`:
- **Подписка** (`admin.subscription`)
- **Версии** (`admin.versions`)
- **Главная страница** (`admin.pages.home`)

#### Модули с `module_connected => true`:
- Документация
- Панель управления
- Ресурсы (Продукты, Категории, Услуги)
- Медиа
- Уведомления
- Пользователи
- Роли
- SEO настройки
- Настройки сайта
- Меню
- Баннеры
- Решения
- Квизы
- Список страниц
- Блоки
- Кейсы
- Модальные окна
- Заявки

### 3. Обновлен компонент Sidebar

**Файл:** `resources/js/components/admin/Sidebar.vue`

#### Изменения:
- Добавлена проверка статуса модуля перед отображением ссылки
- Для нереализованных модулей (`module_connected === false`):
  - Ссылка неактивна (не кликабельна)
  - Отображается текст "(модуль не подключен)" рядом с названием
  - Элемент меню имеет пониженную прозрачность (opacity-50)
  - Курсор изменен на `cursor-not-allowed`

#### Визуальные изменения:
- Нереализованные модули отображаются серым цветом
- Текст "(модуль не подключен)" отображается курсивом и меньшим шрифтом
- При наведении на нереализованный модуль не происходит изменения стиля

## Использование

### Для администраторов

1. **Просмотр документации по SEO:**
   - Перейдите в раздел "Документация"
   - Найдите раздел "SEO оптимизация" в боковом меню
   - Изучите инструкции по настройке SEO

2. **Идентификация нереализованных модулей:**
   - В меню админ-панели нереализованные модули отображаются с пометкой "(модуль не подключен)"
   - Такие модули неактивны и не ведут на страницы

### Для разработчиков

#### Добавление нового модуля:

1. В `app/Services/AdminMenu.php` добавьте пункт меню с полем:
```php
[
    'title' => 'Название модуля',
    'route' => 'admin.module-name',
    'icon' => 'icon-name',
    'roles' => ['admin', 'manager'],
    'module_connected' => true, // или false, если модуль не реализован
]
```

2. Если модуль не реализован, установите `'module_connected' => false`

3. Sidebar автоматически обработает статус модуля

#### Обновление статуса модуля:

Когда модуль будет реализован, измените в `AdminMenu.php`:
```php
'module_connected' => false, // было
'module_connected' => true,  // стало
```
